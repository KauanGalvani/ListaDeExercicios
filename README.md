# Lista de Exercícios – Academia do Programador

## Este repositório contém uma lista de 23 atividades desenvolvidas durante o curso **Academia do Programador**.

Os exercícios foram criados com o objetivo de praticar e consolidar conceitos fundamentais de programação, abordando lógica, estruturas básicas e resolução de problemas.

Cada atividade foi implementada de forma simples e objetiva, servindo como base para o desenvolvimento do raciocínio lógico e evolução nas habilidades de programação.

**Este material representa minha prática e aprendizado ao longo do curso**
